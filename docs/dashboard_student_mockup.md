## 📐 Structure Détaillée

### 1. NAVBAR (Hauteur: 64px)
```
┌────────────────────────────────────────────────────────────────┐
│ Logo    Recherche              Notifs  Messages  Avatar        │
│ 🅼       [🔍____________]        🔔(3)    💬(5)    👤 John ▼    │
└────────────────────────────────────────────────────────────────┘
```

**Éléments** :
- Logo MentorXHub (cliquable → retour dashboard)
- Barre de recherche centrale (chercher mentors)
- Icône notifications avec badge (3 non lues)
- Icône messages avec badge (5 non lus)
- Avatar + nom + dropdown (Mon profil, Paramètres, Déconnexion)

**Interactions** :
- Recherche : HTMX autocomplete en temps réel
- Notifications : Dropdown avec liste des 5 dernières
- Messages : Redirect vers la messagerie
- Avatar : Menu dropdown

---

### 2. SIDEBAR GAUCHE (Largeur: 20%)
```
┌─────────────────┐
│ ● Dashboard     │  ← Actif (bleu)
│   Mes Sessions  │
│   Trouver       │
│   Messages      │
│   Mon Profil    │
│   Paramètres    │
└─────────────────┘
```

**Navigation** :
- **Dashboard** : Vue d'ensemble (page actuelle)
- **Mes Sessions** : Historique et à venir
- **Trouver** : Recherche de mentors
- **Messages** : Messagerie
- **Mon Profil** : Éditer profil
- **Paramètres** : Préférences

**Design** :
- Item actif : Fond bleu clair, texte bleu foncé, bordure gauche bleue
- Item hover : Fond gris clair
- Icônes à gauche de chaque item

#### B. Section "Prochaines Sessions"
```
┌──────────────────────────────────────────────┐
│ 📅 PROCHAINES SESSIONS (2)                   │
└──────────────────────────────────────────────┘

┌──────────────────────────────────────────────┐
│ 👨‍💼 Dr. Sarah Martin                          │
│ Développement Web Fullstack                  │
│ 📆 Demain, 14:00 - 15:00  💻 Visioconférence │
│ [ Rejoindre ]  [ Annuler ]                   │
└──────────────────────────────────────────────┘
```

**Carte de Session** :
- Photo + Nom du mentor
- Sujet de la session
- Date/heure + Mode (visio/présentiel)
- 2 boutons :
  - **Rejoindre** (vert, si dans les 15min avant)
  - **Annuler** (rouge outline)

**Interactions** :
- Rejoindre : Ouvre la visio (Zoom/Google Meet)
- Annuler : Modal de confirmation

## 💡 Interactions UX

### Microinteractions
- **Hover sur carte** : Légère élévation (shadow)
- **Clic sur mentor** : Slide vers page de profil
- **Notification** : Badge animé (pulse)
- **Progression** : Animation du cercle au chargement

### Chargement
- Skeleton screens pour les cartes
- Lazy loading pour les images de profil
- HTMX pour les recommandations

## 🚀 Technologies

### Frontend
- **HTMX** : Chargement dynamique des recommandations
- **Alpine.js** : Dropdowns, modals
- **Chart.js** : Graphique de progression

### Backend
- **Django** : Templates + vues
- **PostgreSQL** : Données
- **Celery** : Notifications en temps réel
